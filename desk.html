Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM:getElementById-Selects by ID.getElementsByClassName -Selects by class.getElementsByTagName-Selects by tag. querySelector-Selects the first match. querySelectorAll -Selects all matches.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Shows and sets HTML tags and text inside an element. innerText: Shows and sets only visible text (hides hidden text). textContent: Shows and sets all text, even hidden ones, without caring about styles or tags.

Question-3: What is event delegation in the DOM?

Event delegation means adding one event listener to a parent element instead of many to each child. When a child is clicked, the event bubbles up to the parent. This makes code faster and easier to manage, especially when adding or removing child elements dynamically.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the clicked element (target) and moves up through its parent elements all the way to the top (the document). This allows parent elements to react to events triggered by their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: Use document.createElement(). Add: Use appendChild() or append() to add to a parent element. Remove: Use removeChild() or remove() to delete an element.
